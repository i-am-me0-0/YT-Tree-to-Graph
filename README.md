# YouTube Adventure Graph Visualizer

An interactive graph visualizer for choose-your-own-adventure stories.

## Featured

- **In Space with Markiplier**
- **A Date with Markiplier**  
- **A Heist with Markiplier**

⚠️ **Spoiler Warning:** If you haven't experienced these stories yourself yet, you really should before looking at all possibilities with this page! Each adventure is best enjoyed first-hand without knowing all the paths ahead of time.

## Backend (Optional)

A Python backend for editing and managing graph data is available in `local-app/backend/`. Run `app.py` to start the full application with server-side editing capabilities.
